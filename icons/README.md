# 图标说明

此目录需要包含以下图标文件：

- icon16.png (16x16 像素)
- icon48.png (48x48 像素)
- icon128.png (128x128 像素)

图标设计建议：
- 使用蓝色主题（#2196F3）
- 显示问号符号（?）
- 圆角矩形背景

可以使用 icon.svg 作为参考，将 SVG 转换为 PNG 格式。

转换方法：
1. 在线工具：https://cloudconvert.com/svg-to-png
2. 使用 ImageMagick: convert icon.svg -resize 16x16 icon16.png
3. 使用 Figma 或其他设计工具导出
